Abstract

This project explores a modern childproofing solution designed to reduce household injuries involving cabinets, sharp objects, and tip-over hazards. NexLock One Touch One Touch aims to provide a secure yet simple locking system for families.

User Research Insights

Parents prioritize ease of use, simple installation, and reliability over cost. Many parents rely on frequent checking instead of trusting current childproofing solutions, revealing flaws in existing products rather than lack of responsibility.

Ideating the Solution

After identifying key safety gaps in existing childproofing products, we developed and evaluated three potential design solutions.

Potential Solution #1

Mechanical Rack & Pinion Lock

A motorized rack-and-pinion system that uses claw-shaped hooks to secure cabinet doors.

How It Works

  • Gear rolls along an internal rack system
  • Hooks extend to lock the cabinet shut
  • Internal motor controls the mechanism
  • Buzzer alerts if the cabinet remains open

Strengths

  • Strong mechanical hold
  • Durable components
  • Clear locking mechanism

Limitations

  • Bulky internal system
  • More complex installation
  • Higher production cost

Potential Solution #2

Electromagnet Mount System

Selected Final Design

Two compact internal electromagnet mounts automatically secure cabinet doors when closed.

How It Works

  • Magnets lock automatically when the cabinet shuts
  • Studs align to increase stability
  • External mounted button releases the magnet
  • Buzzer alerts if the cabinet stays open

Strengths

  • Compact and hidden
  • Simple installation
  • Lower cost components
  • Automatically locks
  • One-touch unlock

Limitations

  • Requires a power source
  • Must ensure magnet strength is reliable

Potential Solution #3

Biometric Deadbolt System

A fingerprint-activated deadbolt system powered by a motor.

How It Works

  • Fingerprint scan verifies the user
  • Motor rotates the deadbolt
  • Door unlocks if authenticated

Strengths

  • Personalized access
  • High-tech appeal
  • Secure authentication

Limitations

  • Expensive components
  • Over-engineered for cabinets
  • More complicated installation
